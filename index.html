Summary

I am an Italian computer scientist and bioinformatician, with a passion for data engineering. In the beginning of my career I got a solid basis in the ITC working as a software developer. After completing my studies in computer science, I moved to Sweden where I got a master degree in bioinformatics. During my master studies, I got interested in large-scale machine learning applications for life science. Thus, I continued my studies in Sweden with a PhD programme in scientific computing (in which I am currently enrolled). In addition to machine learning applications, I am investigating how state-of-the-art data engineering approaches can be applied to data-intensive applications in science. These include cloud computing, virtualization and containerization, GPU acceleration and MapReduce/Dataflow processing models.

Phd student in scientific computing, uppsala university, sweden

oct 2015 ‚Äì present

My PhD focuses on developing cloud-centric methods for life science. As part of the PhenoMeNal Horizon 2020 project (phenomenal.eu), I investigated how the microservice-oriented architecture can be applied to run on-demand computational pipelines on any cloud provider, also by leveraging on application container engines, orchestration platforms and MapReduce frameworks (such as Docker, Kubernetes and Apache Spark). In addition, I have developed a distributed and a GPU-accelerated implementation of the conformal prediction framework (with Apache Spark and Tensorflow). The conformal prediction framework provides object-specific confidence metrics for machine learning predictions, and my implementations have enabled use cases in cheminformatics and in AI-assisted pathology of prostate biopsies.

cooperation associate, cern, switzerland

jun 2018 ‚Äì nov 2018

As CERN cooperation associate I took part in the Helix Nebula Science Cloud (https://hnscicloud.eu) pilot phase. My role consisted in evaluating cloud-based GPU resources provided by the consortium, by running a deep learning application in my field of study. This resulted in a GPU-accelerated implementation of the conformal prediction framework, which enabled us to automatically evaluate a large-scale prostate biopsies dataset from over 80K men in Sweden. For more details please watch my presentation at CERN: https://cds.cern.ch/record/2651149.
